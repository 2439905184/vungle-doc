广告投放素材要求

Ian Harris
2021年04月29日 05:51
关注

广告主需要准备以下素材

视频

尺寸

横版 | 16:9 | 1920 x 1080px | 1280 x 720px  

竖版 | 9:16 | 1080 x 1920px | 720 x 1280px

方版 | 1:1 | 1080 x 1080px | 720 x 720px

时长

15 - 45 秒   

* 按照标准, 我们通常制作的视频时长为15s, 30s, 以及45s

文件大小

小于 60MB

格式

MP4, MOV

 

MREC 视频

尺寸

 6:5 | 300 x 250px | 600 x 500px

时长

5 - 15 秒

文件大小

小于 5MB

格式

MP4, MOV

 

Banner

尺寸

6:1 | 640 x 100px | 320 x 50px | 300 x 50px

8:1 | 800 x 100px | 728 x 90px | 720 x 90px 

请在图片左右两侧预留20px的空白区域

文件大小

6:1 小于60KB I 8:1 小于100KB

格式

PNG, JPG, GIF

 

Video Banner

尺寸

320 x 50px | 300 x 50px | 400 x 50px

*请在video banner的左右两边留白20px

时长

3-10秒

文件大小

2MB之下

文件格式

MP4, MOV

 

Vungle 高级支持

视频 | 可互动结尾卡 | 可试玩结尾卡 | 沉浸式视频结束页

为了制作出最大程度上展示应用/游戏、并能够吸引优质用户的素材，我们希望您可以尽可能多地按照以下要求提供相关文件

File type

  Examples

图片文件

PSD, AI, PNG, JPG
  • 游戏角色
  • 用户界面设计
  • 游戏块 ( 例如. 拼图方块 )
  • 游戏胜利图标或信息 ( 例如. "恭喜!" )
  • CTA按键, 文字 ( 例如. "现在下载", "现在试玩" )
  • 背景图片
  • 启动画面
  • 商店截图
  • 广告宣传图片

视频文件

MOV, MP4
  • 游戏玩法视频素材 ( 关掉背景音乐 打开音效会比较好 )
  • APP试用视频素材
  • 宣传性视频、广告

   *帧率: 建议30fps

动画文件

精灵表单, PNG序列图, 带透明通道的MOV文件
  • 人物或实物的动画
  • 微粒效果动画

3D 文件

C4D, FBX
  • 角色, 实物
  • 不同游戏阶段的地图
  • 武器

   *All 3D project files should be reduced consolidated/collected with relevant texture files

音频文件

WAV, MP3, OGG
  • 游戏音乐
  • 声音效果

字体文件

TTF, OTF
  • 游戏用户界面字体
  • 市场推广字体

其他
  • After Effects(.aep) project files with related image and video assets (reduced and collected)
  • 解锁版游戏 (IPA/APK)
  • 内部/外部建立的视频或可试玩视频
  • Any equirectangular(monoscopic or stereoscopic) or cubemap video and image (for Immersive Video Endcard)

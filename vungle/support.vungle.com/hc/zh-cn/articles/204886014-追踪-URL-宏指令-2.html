追踪 URL 宏指令

Vungle Agent
2021年07月16日 15:22
关注

以下为 Vungle 可以发送给第三方追踪提供商的宏指令清单。标为*必要  的参数对于正确回传至属性必不可少。

Vungle 宏指令

{{{aaid}}} *必要 Google 的广告客户播放服务标识符，设备 AAID 的小写版本（若有）
{{{ad_tracking_disabled}}} 若启用‘请勿追踪’
{{{ad_tracking_enabled}}} 若禁用‘请勿追踪’
{{{age}}} 设备的用户年龄
{{{app_id}}} *必要 广告客户应用商店 id
{{{bid_currency}}} 始终是"USD"
{{{bid_type}}} 活动率类型（查看/安装/完成）
{{{bid_price}}} 表示投标值的浮动值
{{{browser_user_agent}}} 设备的浏览器用户代理
{{{campaign_id}}} 该请求的交付活动 ID
{{{campaign_name}}} 该请求的交付活动名称
{{{carrier_name}}} 请求时的载体
{{{city}}} 设备所在城市
{{{connection_type}}} 连接类型（wifi 或移动）
{{{country}}} 2 个字母的国家代码
{{{country_name}}} 标准国家名称
{{{creative}}} 该请求的交付创意名称
{{{creative_id}}} 该请求的交付创意 ID
{{{device_android}}} 基于设备型号和尺寸的一部 [ipod、phone 或平板电脑]
{{{device_id}}} 设备 ID 的小写版本
{{{device_ios}}} 基于设备 OS 和型号的一部 [iphone、ipad 或 ipod]
{{{device_model}}} 设备型号（如 Galaxy S1）
{{{device_platform}}} 设备 OS （iOS、windows 或 android）
{{{device_windows}}} 基于设备型号和尺寸的一部 [ipod、phone 或平板电脑]
{{{eid}}} 广告服务器/投标方的事件 Id
{{{gender}}} 设备的用户性别
{{{id}}} *必要 见eid
{{{ifa}}} *必要 设备 IFA 的小写版本（若有）
{{{ip}}} 设备 IP
{{{is_coppa_compliant}}} 若发行商设定为符合 COPPA
{{{isu}}} 设备 ISU 的小写版本（若有）
{{{language}}} ISO 639-1 格式的设备语言
{{{latitude}}} 设备纬度
{{{longitude}}} 设备经度
{{{market_id}}} 发行商绑定 ID
{{{max_duration}}} 发行商要求最大视频时长
{{{min_duration}}} 发行商要求最小视频时长
{{{os_version}}} 设备 OS 版本 (10.0.1/9.2.0)
{{{pub_app_id}}} Vungle 内部发行商应用 ID
{{{pub_app_market_id}}} 发行商在应用商店的市场 ID
{{{pub_app_name}}} 发行商应用名称
{{{pub_app_store_category}}} 发行商应用类别（从控制板选择）
{{{pub_app_store_url}}} 发行商应用商店 URL
{{{region}}} 设备所在地区
{{{replaceable_timestamp}}} 见timestamp （用于内部 SDK）
{{{screen_height}}} 设备屏幕高度
{{{screen_orientation}}} 基于宽度/高度，一幅[风景画或肖像]
{{{screen_width}}} 设备屏幕宽度
{{{site_id}}} 见vungle_id
{{{site_name}}} 发行商应用名称
{{{sound_enabled}}} 设备是否静音（是/否）
{{{timestamp}}} 交付的时间戳，格式 "YYYY-MM-DDTHH:mm:SS"
{{{timezone}}} 设备所在时区的标准名称
{{{trackable_device_id}}} 设备 ID 的小写版本，若禁用‘请勿追踪’
{{{user_agent}}} Vungle SDK 用户代理（如 VungleDroid/2.0.1）
{{{video_filename}}} 已交付广告的视频文件名
{{{volume}}} 请求时的设备音量（介于 0-1 的浮动值）
{{{vungle_id}}} Vungle 内部发行商应用 ID

如果您需要更多参数或帮助，请联系tech-support@vungle.com

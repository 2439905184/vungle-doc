URL 매크로 추적

Vungle Agent
2021년 07월 16일 15:22
팔로우

Vungle이 타사 추적 공급자에게 보낼 수 있는 매크로 목록은 다음과 같습니다. *필수 표시가 있는 매개 변수는 포스트 백에서 올바르게 특성화하는 데 필요합니다.

Vungle 매크로

{{{aaid}}} *필수 광고주의 Google Play 서비스 식별자, 소문자 버전의 기기 AAID(제공되는 경우))
{{{ad_tracking_disabled}}} '추적하지 않음'이 활성화된 경우
{{{ad_tracking_enabled}}} '추적하지 않음'이 비활성화된 경우
{{{age}}} 장치 사용자 연령
{{{app_id}}} *필수 광고주 애플리케이션 스토어 ID
{{{bid_currency}}} 항상 미국 달러
{{{bid_type}}} 캠페인 요율 유형(보기/설치/완료)
{{{bid_price}}} 입찰가를 나타내는 float
{{{browser_user_agent}}} 장치의 브라우저 사용자 에이전트
{{{campaign_id}}} 이 요청에 대한 게재 캠페인 ID
{{{campaign_name}}} 이 요청에 대한 게재 캠페인 이름
{{{carrier_name}}} 요청 당시 이동통신사
{{{city}}} 장치 도시
{{{connection_type}}} 연결 유형(WiFi, 모바일)
{{{country}}} 국가 코드 2자리
{{{country_name}}} 표준 국가명
{{{creative}}} 이 요청에 대한 게재 광고 소재 이름
{{{creative_id}}} 이 요청에 대한 게재 광고 소재 ID
{{{device_android}}} 기기 제조업체/모델 및 크기 기준 [ipod, phone, tablet] 중 하나
{{{device_id}}} 장치 ID 소문자 버전
{{{device_ios}}} 기기 OS 및 모델 기준 [iphone, ipad, ipod] 중 하나
{{{device_model}}} 장치 모델(예: 갤럭시 S1)
{{{device_platform}}} 장치 OS(iOS, windows, android)
{{{device_windows}}} 기기 제조업체/모델 및 크기 기준 [ipod, phone, tablet] 중 하나
{{{eid}}} 광고 서버/입찰자 측 이벤트 ID
{{{gender}}} 장치 사용자 성별
{{{id}}} *필수 eid와 동일
{{{ifa}}} *필수 장치 IFA 소문자 버전(제공되는 경우)
{{{ip}}} 장치 IP
{{{is_coppa_compliant}}} 게시자가 COPPA를 준수하도록 설정된 경우
{{{isu}}} 장치 ISU 소문자 버전(제공되는 경우)
{{{language}}} ISO 639-1 포멧의 장치 언어
{{{latitude}}} 장치 위도
{{{longitude}}} 장치 경도
{{{market_id}}} 퍼블리셔 번들 ID
{{{max_duration}}} 퍼블리셔가 요청한 비디오 최대 길이
{{{min_duration}}} 퍼블리셔가 요청한 비디오 최소 길이
{{{os_version}}} 장치 OS 버전(10.0.1/9.2.0)
{{{pub_app_id}}} Vungle 내부 퍼블리셔 앱 ID
{{{pub_app_market_id}}} 퍼블리셔의 앱 스토어 마켓 ID
{{{pub_app_name}}} 퍼블리셔 앱 이름
{{{pub_app_store_category}}} 퍼블리셔 앱 장르(대시보드에서)
{{{pub_app_store_url}}} 퍼블리셔 앱 스토어 URL
{{{region}}} 장치 지역
{{{replaceable_timestamp}}} timestamp와 동일(내부 SDK용)
{{{screen_height}}} 장치 화면 높이
{{{screen_orientation}}} 폭/높이 기준, [landscape, portrait] 중 하나
{{{screen_width}}} 장치 화면 폭
{{{site_id}}} vungle_id와 동일
{{{site_name}}} 퍼블리셔 앱 이름
{{{sound_enabled}}} 장치 음소거 여부(true/false)
{{{timestamp}}} "YYYY-MM-DDTHH:mm:SS" 포멧의 게재 타임스탬프
{{{timezone}}} 표준 장치 시간대 이름
{{{trackable_device_id}}} 장치 ID 소문자 버전('추적하지 않음'이 활성화되지 않은 경우)
{{{user_agent}}} Vungle SDK 사용자 에이전트(예: VungleDroid/2.0.1)
{{{video_filename}}} 게재된 광고의 비디오 파일 이름
{{{volume}}} 요청 당시 장치 볼륨(0-1 사이의 부동 소수점)
{{{vungle_id}}} Vungle 내부 퍼블리셔 앱 ID

추가 매개 변수가 필요하거나 도움이 필요한 경우 tech-support@vungle.com으로 연락하시기 바랍니다.

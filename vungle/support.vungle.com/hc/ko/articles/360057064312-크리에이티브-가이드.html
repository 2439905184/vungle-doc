퀄리티 유저 유입 및 게임을 효율적으로 보여주기 위해, 가능한 많은 에셋을 공유해 주세요.

File type

예시

이미지 파일

PSD, AI, PNG, JPG
  • Game Characters
  • UI Design
  • Game Pieces (ex. puzzle block)
  • Winning Icons/Messaging (ex. "Download now", "Play now")
  • CTA button, text
  • Background Images
  • Splash screens
  • Store screenshots
  • Marketing images, ads

비디오 파일

MOV, MP4
  • Gameplay footage
    (Background music OFF, Sound effects ON is better)
  • App experience footage
  • Marketing video, ads

   *Frame Rate: 30fps recommended

애니메이션 파일

Sprite sheets,

PNG sequences,

MOV with alpha channel
  • Animation of Characters, Objects
  • Particle Effect Animations

3D 파일

C4D, FBX
  • Characters - ideally with animation baked & references merged
  • Objects
  • Maps from game levels
  • Weapons

   *All 3D project files should be reduced consolidated/collected with relevant texture files

오디오 파일

WAV, MP3, OGG
  • Game music
  • Sound effects

폰트 files

TTF, OTF
  • Game UI fonts
  • Marketing fonts

기타
  • After Effects(.aep) project files with related image and video assets (reduced and collected)
  • Unlocked game (IPA/APK)
  • Internally/externally built videos, playables.
  • Any equirectangular(monoscopic or stereoscopic) or cubemap video and image (for Immersive Video Endcard)
